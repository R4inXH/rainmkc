UI/UX

UX Writer

Jenis pekerjaan dalam UI UX yang pertama adalah UX writer. UX writer merupakan seseorang yang memiliki tuas dalam membuat sebuah teks copy pada website, aplikasi, ataupun produk digital lainnya.

UI Designer

UI designer memiliki tugas untuk menentukan tampilan halaman web atau aplikasi secara keseluruhan. UI design memiliki tanggungjawab untuk mendesain web dari pemilihan warna, gambar, tulisan, dan lainnya agar lebih menarik.

UX Designer

UX designer merupakan seseorang yang bertanggungjawab atas peningkatan kepuasan penggunaan produk.
UX designer akan memikirkan cara bagaimana pengguna akan mudah dan nyaman dalam menggunakan suatu produk.

UX Researcher

Jenis pekerjaan UI UX yang satu ini merupakan profesi yang eksis dalam bidang teknologi terutama pada perusahaan e-commerce ataupun marketplace.
Tugas utama dari seorang UX researcher adalah memberikan jawaban atas semua pertanyaan yang paling challenging dalam desain suatu produk.

UX Engineer

UX engineer adalah profesi yang mempunyai peran dalam mengidentifikasi kebiasaan dari pengguna data melakukan interaksi dengan suatu elemen desain.
Profesi ini juga dapat dikatakan sebagai penghubung antara designer dengan development.

Cyber security (dalam Negeri)
tipkerja.com

Di Indonesia, seorang cyber security dapat menerima gaji sebesar Rp. 6.500.000 – 12.000.000 dengan background pendidikan S1 sederajat. Cyber security dapat memperoleh gaji lebih besar apabila memiliki sertifikasi profesi seperti CISM (Certified Information Security Manager) dan CISSP (Certified Information System Security Professional).
